## 总览

`Algorithm-Swift` 是一套使用Swift语言编码的常用算法。

* Red-Black Tree (红黑树)
* Dijkstra (最短路径算法)
* Genetic Algorithm (遗传算法)

## 使用

直接将需要的算法代码文件Copy到你的项目中即可。
